About

I started this project for my senior graphic design thesis at the University of Florida after spending a lot of time in NYC and wishing a comprehensive resource like this existed.

Some museums have suggested admission. Please support them if you're able to!


All data is sourced by me, and I do my best to keep it up to date. If anything is incorrect, please fill out the issue tracker here. To request a feature, fill out the feature request here. For anything else (or just to talk maps), email me!


Huge thank you to MapLibre and OpenFreeMap for keeping this project free, as well as freeCodeCamp for teaching me to code.
